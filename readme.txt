Steps:

Run the main.py file using the command "python main.py"
Alternatively you can also run the main.exe file.

==========================================================================================

Requirements:

Python - 3.6.1 (64 bit)
dash - 0.21.0
	pip install dash==0.21.0	
dash-renderer - 0.12.1
	pip install dash-renderer==0.12.1
dash-html-components - 0.10.0
	pip install dash-html-components==0.10.0
dash-core-components - 0.13.0-rc4 (This is a pre release version which supports Tabs. The current version doesn't)
	pip install dash-core-components==0.13.0-rc4
plotly - 2.5.1
	pip install plotly==2.5.1
